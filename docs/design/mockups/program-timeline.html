Key Design Decisions

Journey header — name, dates, week number, and days-to-go countdown. Instant context on where they are in time.

Phase timeline is proportional — wider phases represent longer time spans. Current phase highlighted in yellow. Completed phases in green. Future phases muted. Phase transition gates shown below with their specific requirements.

Milestones track both achieved (green) and upcoming goals. Gives athletes the "look how far I've come" motivation alongside "here's what's next" direction.

Top-10 priorities show first 5 with expand for rest. Each priority shows which coach owns it and a one-line rationale. Priority #1 gets a red alert border if it's a critical issue (like the sleep crisis). Display only — athlete raises concerns through check-ins.

Coaching team grid — universal coaches (dark border) vs goal-specific (blue border). Each card shows slot, type, name, domain, and a link to the full profile. "View Full Profile" opens a detail view with the complete persona, diagnostic rules, and conflict resolution priority.

Request Reassessment button triggers the mid-journey intake flow. This is how athletes initiate changes outside the regular check-in cycle.
