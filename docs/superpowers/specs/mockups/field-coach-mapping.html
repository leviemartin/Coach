Candidate New Fields

These are currently only captured weekly in checkin, but daily tracking would give coaches richer pattern data.

+

Pain / Injury Flag (0-3 scale)

Currently: Baker's Cyst (0-10) and Lower Back (0-10) only at weekly checkin
Proposed: Simple daily flag: 0 = nothing, 1 = mild, 2 = moderate, 3 = need to stop. Optional body area tag.
Coach value: Mobility agent sees "knee pain spiked Tue-Wed after Monday leg session, resolved Thursday" instead of "Baker's Cyst: 3/10 this week"

+

Energy Level (1-5)

Currently: Perceived readiness only at weekly checkin (1-5 scale)
Proposed: Quick daily tap: 1 = wrecked, 3 = normal, 5 = great
Coach value: Recovery agent sees daily energy pattern vs. Garmin Body Battery. Mental Performance agent tracks motivation trends. "Energy crashed Wed-Thu after bad sleep Tue night."

+

Sleep Disruption Tag

Currently: Only captured in free-text notes ("kid up 3x")
Proposed: Quick tag selection: None, Kid(s) woke up, Stress/mind racing, Pain, Other
Coach value: Recovery agent differentiates behavioral sleep issues (fixable) from external disruptions (adapt around). "3 of 5 bad sleep nights were kid-related, not bedtime compliance failures."

+

Nutrition Hit Target

Currently: Only Garmin nutrition stats (often incomplete/unreliable)
Proposed: Simple daily yes/no: "Hit protein target today?"
Coach value: Nutrition agent gets real compliance data instead of unreliable Garmin nutrition sync. Pairs with kitchen cutoff for full daily nutrition picture.

+

Structured Notes (Replace Free Text)

Currently: Single free-text field — everything goes in one box
Proposed: Optional tagged notes: select a category (Injury, Sleep, Training, Life/Stress, Other) then type. Multiple notes per day allowed.
Coach value: Agents can filter by category. Mobility agent sees only injury notes. Recovery agent sees sleep notes. Mental Performance sees stress notes. No more signal buried in noise.
